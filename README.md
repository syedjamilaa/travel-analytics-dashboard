# ✈️ AI-Powered Travel Analytics Dashboard

> Ask any business question in plain English. Get SQL, charts, 
> and AI insights — instantly.

## 🔗 Live Demo
**[→ Open Dashboard](https://syed-travel-analytics.streamlit.app/)**

## 🎯 What This Project Demonstrates
- End-to-end data pipeline (generation → storage → query → visualisation)
- AI integration (Google Gemini for NL→SQL and insight generation)
- Product thinking (auto chart detection, safety validation, UX for non-technical users)
- Deployment (Streamlit Cloud with secrets management)

## 📊 Dataset
100,000+ rows across 4 tables with realistic business patterns:
| Table | Rows | Purpose |
|-------|------|---------|
| users | 5,000 | Demographics, device, user type |
| properties | 1,000 | Destinations, ratings, pricing |
| bookings | 20,000 | Transactions, status, revenue |
| events | ~80,000 | Funnel tracking (search→view→cart→book) |
